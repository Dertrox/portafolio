Sistema de Telemetría para Cohetes Experimentales

Desarrollo de un sistema electrónico para adquisición, almacenamiento y transmisión de datos en tiempo real durante pruebas de vuelo de cohetes experimentales.

ESP32 LoRa KiCad SPI I2C UART

Objetivo

Diseñar un sistema capaz de capturar múltiples variables físicas, almacenarlas localmente y transmitirlas en tiempo real, garantizando estabilidad eléctrica y confiabilidad en condiciones reales.

Arquitectura del sistema

El sistema está compuesto por:

  • Microcontrolador principal (ESP32)
  • Sensores conectados vía I2C
  • Memoria microSD (SPI)
  • Módulo LoRa para transmisión
  • Regulación de voltaje (5V / 3.3V)

Prototipo 1

Se implementó un diseño inicial con reguladores lineales (LM7805 y LM117) y comunicación mediante buses SPI, I2C y UART.

Este prototipo permitió validar:

  • Lectura simultánea de sensores
  • Transmisión LoRa
  • Registro en microSD

Problemas detectados

  • Baja calidad de ruteo en capa inferior
  • Fallas en comunicación SPI con microSD
  • Fragilidad mecánica

Estas limitaciones comprometían la confiabilidad del sistema, especialmente en el almacenamiento de datos.

Prototipo 2 – Rediseño estructural

Se rediseñó el sistema separando la electrónica en dos placas:

  • Placa A: Sensores y adquisición
  • Placa B: Control y potencia

Esta arquitectura permitió:

  • Mejor organización de buses (I2C, SPI, UART)
  • Reducción de interferencias
  • Mayor estabilidad eléctrica

Mejoras implementadas

  • Desacoplos adicionales en alimentación
  • Distribución más clara de señales
  • Mayor robustez estructural

Problema crítico: disipación térmica

Durante pruebas prolongadas, los reguladores lineales presentaron sobrecalentamiento, especialmente durante:

  • Transmisión LoRa
  • Escritura en microSD

La disipación de potencia excedía la capacidad térmica del encapsulado, provocando apagados del sistema.

Solución implementada

Se reemplazó la regulación lineal por un convertidor DC-DC basado en LM2596.

  • Mayor eficiencia energética
  • Reducción de temperatura
  • Mayor estabilidad del sistema

Resultado

El sistema final logró una operación más estable, con mejor distribución de energía y mayor confiabilidad en la adquisición y transmisión de datos.
