Proyecto destacado

Sistema de Telemetría para Cohetes Experimentales

Desarrollo de un sistema completo para adquisición, almacenamiento, transmisión y visualización de datos en tiempo real, con una placa final fabricada en JLCPCB y un flujo de trabajo que integra firmware, dashboard y simulación.

ESP32-S3 LoRa SX1262 KiCad JLCPCB Python OpenRocket MATLAB FreeRTOS

Placa final

Esta es la versión final del sistema: una placa diseñada en KiCad y fabricada en JLCPCB, integrada para operar como computadora de vuelo con sensado, comunicaciones, almacenamiento y control.

Placa final Fabricación industrial Integración completa

Arquitectura del sistema

Unidad de vuelo

Computadora de vuelo basada en ESP32-S3 con LoRa SX1262, encargada de leer sensores, registrar datos y transmitir telemetría hacia tierra.

Estación base

Nodo receptor conectado al computador para recibir tramas, validarlas y mostrarlas en el dashboard.

Componentes principales

Sensado

BME680 para presión, temperatura y humedad. MPU6050 para aceleración y velocidad angular. GPS GY-NEO6MV2 para posición y seguimiento.

Comunicación

LoRa SX1262 a 915 MHz para enlace de largo alcance con enfoque en telemetría experimental.

Registro y energía

microSD para almacenamiento local y etapa de potencia con conversión DC-DC para mejorar eficiencia y estabilidad.

Firmware embebido

El firmware organiza el sistema con FreeRTOS para manejar adquisición, radio, servo de recuperación y diagnóstico LED. El diseño lógico separa tareas como sensores, transmisión LoRa, control del servo y estado del sistema.

TaskSensors

Lee BME680, MPU6050 y GPS; calcula variables derivadas; construye la estructura de telemetría.

TaskLoRa

Empaqueta la telemetría y administra la transmisión y recepción del enlace con la estación base.

TaskServo

Gestiona el actuador de recuperación para el despliegue del paracaídas.

TaskLED

Provee diagnóstico visual y estado de fallos en el sistema.

Lógica de vuelo y recuperación

La lógica del sistema define estados de vuelo y usa umbrales calculados a partir de la dinámica del cohete. El despliegue del paracaídas no depende de una sola lectura: se evalúan aceleración, velocidad vertical y altitud relativa para evitar activaciones erróneas.

Dashboard de visualización

Se desarrolló una interfaz en Python y una capa web con HTML, CSS y JavaScript para visualizar la telemetría recibida desde la estación base. El sistema registra datos y permite seguir variables como altitud, temperatura, humedad, posición, RSSI y SNR.

Recepción

Lectura serial de tramas, parseo de datos y actualización en tiempo real.

Registro

Salida a CSV para conservar datos de vuelo y métricas del enlace.

Simulación y validación

OpenRocket

Se usó para modelar el cohete, verificar geometría, estabilidad y comportamiento general del vuelo. Las curvas obtenidas ayudaron a definir el perfil de ascenso y descenso.

MATLAB

Se utilizó para validar y comparar la respuesta de vuelo mediante simulación matemática, cruzando resultados con OpenRocket para definir umbrales y criterios de despliegue.

Resultados

El resultado final es una plataforma de telemetría funcional para cohetería experimental: adquiere datos, los guarda localmente, los transmite por LoRa y los presenta en una interfaz de monitoreo. Además, el diseño quedó listo para pruebas y expansión futura.
