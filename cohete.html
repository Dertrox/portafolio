Sistema de Telemetría para Cohetes Experimentales

Sistema electrónico diseñado para la adquisición, almacenamiento y transmisión de datos en tiempo real durante pruebas de vuelo en cohetes experimentales.

ESP32 LoRa KiCad SPI I2C UART DC-DC

Arquitectura del sistema

  • Placa A: Sensores y adquisición de datos
  • Placa B: Control, procesamiento y potencia

La separación del sistema en dos módulos permitió mejorar la organización de señales, reducir interferencias y aumentar la estabilidad general del sistema.

Funcionamiento

  • Lectura simultánea de sensores mediante bus I2C
  • Registro de datos en memoria microSD mediante SPI
  • Transmisión inalámbrica en tiempo real mediante LoRa
  • Comunicación interna usando UART para control y depuración

Diseño electrónico

El sistema fue diseñado en KiCad, implementando una arquitectura modular que separa adquisición y control. Se aplicaron técnicas de desacoplo en líneas de alimentación y organización de buses para garantizar integridad de señal.

Gestión de energía

Se implementó un sistema de regulación basado en convertidores DC-DC (LM2596), mejorando la eficiencia energética y reduciendo problemas térmicos en condiciones de carga.

Decisiones de diseño

  • Separación física de módulos para reducir interferencias
  • Uso de buses estándar (I2C, SPI, UART) para escalabilidad
  • Implementación de regulación conmutada para eficiencia energética
  • Arquitectura preparada para futuras expansiones del sistema

Resultado

El sistema final permite la adquisición confiable de datos, su almacenamiento local y transmisión en tiempo real, proporcionando una plataforma estable para pruebas experimentales.
